What is Micropub?

Micropub is a W3C standard that specifies methods for creating, updating, and deleting content on your website. Because all Micropub clients and servers are developed in adherence to the same standard, you can use a client to create any type of post that your server can then process in the way you want.

James, the creator of this tool, uses Micropub to create posts that show what coffees he is drinking, upload "likes" to his website, and more.

There are two components to Micropub. First, there is the client. The client lets you create posts through a website or an application. Second, there is the server. The server turns the information you give to the client into a post on your website.

Can I host my own Micropub client and server?

Yes! You can host your own Micropub endpoint. This client and server is open-sourced so you can use this one if you would like. Please note the server part of this application was designed specifically to turn posts into markdown files with front matter. This is the ideal configuration if you choose to use a static generator like Jekyll on your site. You can deploy a version of this Micropub client and server by going to the project GitHub page and following the instructions.

You can find other Micropub clients on the IndieWeb wiki.
