Micropub Endpoint Schemas

This page lists all of the schemas for micropub endpoint requests made from this client.

You can use these schemas to understand how the Micropub client sends requests to a server.

You may find these schemas especially helpful if you use this client but not the built-in server associated with this client.

Reply



      {

         "type":[
      
            "h-entry"
      
         ],
      
         "in-reply-to":[
      
            "https://jamesg.blog"
      
         ],
      
         "properties":{
      
            "content":[
      
               {
      
                  "html":"I love your blog!"
      
               }
      
            ],
      
            "category":[
      
               "Webmention"
      
            ]
      
         }
      
      }

Note



      {

         "type":[
      
            "h-entry"
      
         ],
      
         "properties":{
      
            "title":[
      
               "Micropub is a great way to post content on your site"
      
            ],
      
            "content":[
      
               {
      
                  "html":"The Micropub standard lets me post content to my site from my own client and any other client that is standard-compliant."
      
               }
      
            ],
      
            "category":[
      
               "IndieWeb"
      
            ],
      
            "is_hidden":[
      
               "no"
      
            ]
      
         }
      
      }

RSVP



      {

         "type":[
      
            "h-entry"
      
         ],
      
         "in-reply-to":[
      
            "https://indieweb.org"
      
         ],
      
         "p-rsvp":{
      
            "properties":{
      
               "event_name":"IndieWebCamp",
      
               "in-reply-to":"https://indieweb.org",
      
               "state":"yes",
      
               "content":[
      
                  "I am RSVPing to this event."
      
               ],
      
               "event_date":"2021-01-01",
      
               "event_time":"10:00"
      
            }
      
         }
      
      }
